## PyImageResizer
A simple Python script that will resize images to a given width and height

### Dependencies
[Python Pillow Library](https://pillow.readthedocs.io/en/latest/)

### Options
``` 
--outputdir / -o
-o 'myTestOutputDir'
``` 

``` 
--imageDir / -i
-i 'myImageDir'
``` 

``` 
--width / -w
-w 1000
``` 

### Example
````
./py-resizer -i myImagesDir -o myOutputDir -w 1000
````
